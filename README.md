# scan-apks

## Introduce
方便手机端直接扫码下载apk
扫描当前目录下的apk文件，显示在终端上，及时看到下载信息。
网页端 localhost:3010/

## Usage
1. 将本项目与需要扫描的项目放到同一目录下
2. yarn install
3. yarn start

## usage 2
1. yarn install
2. yarn pkg 打包exe
3. scan-apks.exe 应用程序 移动至目标目录的父级目录

## Tips
1. 默认设置3010端口
2. 终端显示下载二维码
3. quickStart.bat 一键启动，可指定项目目录位置
